1. Программный код - текстовый файл
2. Программный код на Java содержится в файле с расширением .java
3. В языке Java ВСЁ является классом. То есть нет открытых функций - всё заключено в классы.
4. Приложение запускается из ТОЧКИ ВХОДА - специальный класс в котором есть метод с сигнатурой:
public static void main (String[] args){.....}
public - публичный спецификатор доступа;
static - статический (то есть метод можно запустить без создания экземпляра класса);
void - возвращаемое значение фунции (void значит - ничего)
Примечание: сигнатура ЛЮБОЙ функции (кроме конструктора)
<Спецификатор доступа public/protected/private> <тип - статическая/не статическая> <тип возвращаемого значения> <имя функции> (<Список аргументов>)
5. Именование файлов, классов, методов и атрибутов
-Имя класса ДОЛЖНО начинаться с заглавной латинской буквы и содержать латинские буквы и цифры
-Имя файла ПОЛНОСТЬЮ совпадает с именем класса, содержащегося в нём и имеет расширение .java
-Имя метода/атрибута ДОЛЖНО начинатся со строчной латинской буквы и содержит латинские буквы и цифры
Пример: Hello.java

Атрибуты и методы
Класс - новая переменная, которую создал пользователь.
Класс содержит в себе:
- Атрибуты - другие переменные;
- Методы - функции, которые работают как с атрибутами класса так и с внешними переменными.
Класс по сути ОБЪЕДИНЯЕТ в себе некоторые значения (атрибуты) и предоставляет возможности по работе с ними (методы).
Класс машина - содержит в себе атрибуты:
Цвет (строка)
Модель (строка)
Объём двигателя (действительное число)
Пример: Car.java
Методы - функции для работы с классом.
Класс нужно использовать - то есть задавать значения атрибутам и вызывать методы.
Обычно приложение состоит из большого числа классов и одного класса - точки входа.

Задача №1: написать класс Точка на плоскости. Он должен содержать в себе атрибуты - координаты точки и метод - выводит на экран номер четверти координатной плоскости в которой она содержится или ось на которой она лежит

Задача №2: написать класс Прямая - содержит два атрибута (точки) - начало и конец
Содержит функцию - вычисление длины прямой.

Задача №3: написать класс Прямоугольник - содержит в себе два атрибута - Прямые (объекты классов из задачи №2), или 4 точки.
Содержит функции:
а) вычисление периметра
б) вычисление площади

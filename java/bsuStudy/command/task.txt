Идея:
Разработать API для Пульта умного дома.
Пульт содержит 7 кнопок.
6 из них - вкл/выкл устройства.
7 - глобальная отмена.
Классы для управления устройствами уже реализованы.

Задача: создать API для программирования пульта. Чтобы каждая ячейка (2 кнопки) могла быть настроена на управление устройством или группой устройств.
Примечание: пульт должен поддерживать не только существующие устройства, но и любые другие, которые будут добавлены в будующем.

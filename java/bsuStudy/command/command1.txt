Пример работы паттерна Команда
- Посетитель кафе
- Официантка
- Заказ
- Повар

1. Посетитель передаёт заказ официантке
2. Официантка оформляет бланк заказа
3. Официантка передаёт заполненный бланк повару
Примечание: заказ содержит все инструкции для его выполнения.
4. Повар выполняет инструкции - готовит содержащиеся в заказе блюда
